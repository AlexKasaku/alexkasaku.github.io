Sitecore.Sweep

Open-source Sitecore module
for automatic cleaning of rogue HTML

Operates as its own pipeline

Called on UI saves

in both Content Editor + Experience Editor

Configuration determines what
templates + fields should be cleaned

Can also just be triggered manually
via the Rich-Text Editor

Included ruleset

Removing inline styling

<p style="margin-top:430px">
    A lovely paragraph
</p>

⬇

<p>A lovely paragraph</p>

Removing unwanted classes

<p class="badclass goodclass">
    A lovely paragraph
</p>

⬇

<p class="goodclass">
    A lovely paragraph
</p>

Supports both whitelisting + blacklisting class names

Removing empty elements

<p>Some content</p>
<p></p>
<p>Some more content</p>

⬇

<p>Some content</p>
<p>Some more content</p>

Fixing poorly made section headers

<p><strong>My title!</strong></p>

⬇

<h2>My title!</h2>

Removing non-breaking spaces

<p>I       like      weird    spaces!</p>

⬇

<p>I like weird spaces!</p>

These are not for styling your site!

+ add your own!
